## Tracking professional qualifications web system for diploma
Web-based system for tracking professional qualifications, training hours, and institutional certificates (Spring Boot/PostgreSQL).

---

## 🚀 Getting Started: Environment Configuration

To get this project up and running on your local machine, you'll need to configure your environment variables. We use a `.env` file to manage sensitive information like API keys, database credentials, and port settings.

### 🛠 Setup Instructions

1. **Locate the template:** Look for the `env.example` file in the root directory. This file contains all the necessary keys the application requires, but without the actual values.
2. **Create your local file:** Copy the contents of `env.example` into a new file named `.env`.
* *Note: The `.env` file is ignored by git to keep your secrets safe.*


3. **Update the values:** Open your new `.env` file and replace the placeholder values with your actual local configuration.

---

### 💻 Quick Command

If you are using a terminal, you can quickly clone the file using one of the following commands:

**On Mac / Linux:**

```bash
cp env.example .env
```

**On Windows (Command Prompt):**

```cmd
copy env.example .env
```

**On Windows (PowerShell):**

```powershell
cp env.example .env
```
